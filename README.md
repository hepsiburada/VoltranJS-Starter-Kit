
# VoltranJS Starter Kit


## Setup & Install


#### For Yarn


```sh
$ yarn install
```

####  For Npm

```sh
$ npm install
```

## Run For Development 

#### For Yarn

```sh
$ yarn start
```

####  For Npm

```sh
$ npm run start
```

## Run For Production

#### For Yarn

```sh
$ yarn serve
```

####  For Npm

```sh
$ npm run serve
```
